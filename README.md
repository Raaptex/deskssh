# DeskSSH
Use SSH Server with Desktop app

## Installation

> **IMPORTANT !** DeskSSH uses google chrome, so it must be installed

_Linux :_

Install python
```
sudo apt-get update
sudo apt-get install python3.6
sudo apt-get install python3-pip
```
Download code
```
git clone https://github.com/Raaptex/deskssh
```

## Start
Linux :
```
sh ./start.sh
```
